A terminal that actually works

Full command set: ls, cd, cat, mkdir, neofetch and more. History navigation, Ctrl+L to clear, open files directly from the shell. JetBrains Mono, always.

The terminal is a first-class citizen — not an afterthought bolted on.

🚀
Boot animation

TanyelOS logo fades in over a dark background. A smooth progress bar. Status messages. The whole OS-startup theatre — done right.

🪟
Draggable windows

Drag, resize, minimize, maximize, close. Traffic-light buttons. Click to bring to front. Double-click titlebar to maximize. It works exactly the way you expect.

📁
File Explorer

Sidebar with places, breadcrumb navigation, grid view with icons. Double-click to open files and folders. Single-click to select.

📋
Taskbar + Start menu

Frosted glass taskbar at the bottom. All open windows listed. Click to restore or minimize. Start button opens a searchable app grid.

🖱️
Right-click menu

Right-click the desktop for quick actions: refresh, change wallpaper, toggle theme, open terminal, open files, display settings.

🖥️
Desktop icons

Click to select with highlight. Double-click to open. Right-click for context. Drag-select multiple icons. All the desktop basics, none of the clutter.

Customise

Make it yours.

Accent color, theme, font, wallpaper — all changeable in Settings without touching a config file.

VirtualBox Free Windows

  1. Download VirtualBox at virtualbox.org
  2. New VM → Linux → Ubuntu 64-bit
  3. 4 GB RAM · 2 CPU · 25 GB disk
  4. Storage → attach the ISO
  5. Display → 128 MB VRAM · VMSVGA
  6. Boot and enjoy

VMware Player Free Windows

  1. Download VMware Workstation Player
  2. New VM → installer disc image → ISO
  3. 4 GB RAM · 2 CPU · 25 GB disk
  4. Enable 3D acceleration in settings
  5. Boot — VMware Tools installs auto

USB Flash Drive

  1. Download Balena Etcher (Windows)
  2. Flash the ISO to a USB drive
  3. Restart PC, boot from USB
  4. Try TanyelOS live — nothing installed
  5. Click "Install" to make it permanent
