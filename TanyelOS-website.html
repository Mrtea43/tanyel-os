Boot animation

TanyelOS logo fades in over a dark background. A smooth teal progress bar. The whole boot theatre — done right.

🪟
Draggable windows

Drag, resize, minimize, maximize, close. Traffic-light buttons. Click to bring to front. Works exactly as you expect.

📁
File explorer

Sidebar with places, breadcrumb nav, grid view. Double-click to open. Single-click to select.

📋
Taskbar + Start menu

Frosted glass taskbar at the bottom. Open windows listed. Start button opens a searchable app grid.

🎨
Live theming

Switch accent color, wallpaper, light/dark mode — all from Settings. No config files touched.

🖥️
Desktop icons

Click to select. Double-click to open. Right-click for context. All the desktop basics, none of the clutter.

VirtualBox Free Windows

  1. Download VirtualBox at virtualbox.org
  2. New VM → Linux → Ubuntu 64-bit
  3. 4 GB RAM · 2 CPU · 25 GB disk
  4. Storage → attach the ISO
  5. Display → 128 MB VRAM · VMSVGA
  6. Boot and enjoy

VMware Player Free Windows

  1. Download VMware Workstation Player
  2. New VM → installer disc image → ISO
  3. 4 GB RAM · 2 CPU · 25 GB disk
  4. Enable 3D acceleration in settings
  5. Boot — VMware Tools installs auto

USB Flash Drive

  1. Download Balena Etcher (Windows)
  2. Flash the ISO to a USB drive
  3. Restart PC, boot from USB
  4. Try TanyelOS live — nothing installed
  5. Click "Install" to make it permanent
